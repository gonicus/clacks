Notify¶

Clacks D-Bus Notification Plugin¶

This plugin allows to notify a single or all users on the client.

class clacks.dbus.plugins.notify.main.DBusNotifyHandler¶

This dbus plugin is able to send notification to the user using the systems D-Bus.

This plugin exports dbus methods that cannot be accessed directly through the clacks-client DBus-Proxy.

For details on how to use the user notification please see: clacks.client.plugins.notify.utils.Notify.
