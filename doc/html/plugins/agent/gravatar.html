Gravatar¶

class clacks.agent.plugins.misc.gravatar.Gravatar¶

Utility class that contains methods needed to retrieve gravatar URLs.

getGravatarURL(mail, size=40, url='http://www.gonicus.de')¶

Note

This method will be exported by the CommandRegistry.

Generate the gravatar URL to be used for user pictures on demand.

Parameter Description
mail Gravatar’s mail address
size desired image size
url Clickable URL

Return: Image URL
