Plugin development¶

Basically there are four plugin types that are used inside of Clacks. Two of them are agent plugins - namely handlers and ordinary plugins, one flavor of client plugins and one flavor of dbus plugins.

In order to help with quick plugin templating, there’s a helper script tools/clacks-plugin-skel which asks a couple of questions and generates a quickstart for you:

$ tools/clacks-plugin-skel
Generate plugin skeleton. Please provide some information:

Plugin name (must be [a-z][a-z0-9]+): sample
Plugin type (agent, client, dbus): agent
Version: 1.0
Authors name: Cajus Pollmeier
Authors email: pollmeier@gonicus.de

Done. Please check out the 'sample' directory.
$

Here’s the resulting directory structure:
