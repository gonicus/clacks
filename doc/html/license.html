History and License¶

History of the software¶

Clacks was created ein the early 2001s by Cajus Pollmeier at GONICUS GmbH (see http://www.gonicus.de/) in Germany. Besides the work done by GONICUS it includes many contributions from others.

All Clacks releases are Open Source (see http://www.opensource.org/ for the Open Source Definition). All Clacks versions have been release under the terms of the GPL v2; the table below summarizes the various releases.

Release Derived from Year Codebase
1.0 thru 1.99 n/a 2001-2003 PHP
2.0 thru 2.4 1.0 2003-2004 PHP
2.5 thru 2.5.13 2.4 2004-2007 PHP
2.6 2.5 2007-2011 PHP/Perl
2.7 2.6/scratch 2011- PHP/Python
