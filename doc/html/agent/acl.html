ACL resolver¶

class gosa.agent.acl.ACL(scope=None, role=None)¶

The ACL class contains list of action for a set of members. These ACL classes can then be bundled and attached to a ldap base using the ACLSet class.

add_action(target, acls, options)¶

Adds a new action to this acl.

add_member(member)¶

Adds a new member to this acl.

add_members(members)¶

Adds a list of new members to this acl.

get_members()¶

Returns the list of members this ACL is valid for.

get_type()¶

Returns the type of an ACL. SUB, PSUB, RESET, ...

match(user, action, acls, options={}, skip_user_check=False)¶

Check of the requested user, action and the action options match this acl-object.

repr_self(indent=0)¶

Generates a human readable representation of the acl-object.

use_role(role)¶

Mark this ACL to use a role instead of direkt permission settings.

class gosa.agent.acl.ACLRole(name)¶

add(item)¶

get_name()¶
